## 🧠 思考框架

> 以下是我（用户）的投资思考习惯和分析框架。AI 在生成报告时应融入这些视角。

### 1. 大资金行为逻辑
- [x] 分析大跌时检查：近期有无重大 IPO/增发？（机构可能为打新出货换仓）
- [x] 大宗交易(block trade)异动可能揭示机构真实意图
- [x] 「官方叙事」（如数据导致加息预期）往往是掩护，真正的资金逻辑可能不同

### 2. 多空博弈思维
- [x] 每个判断必须同时呈现 Bull case 和 Bear case，不允许只说一方
- [x] 给出概率化判断，不骑墙
- [x] 追问「然后呢？」——每个事件的影响链不只有一层

### 3. 大V信号客观解读
- [x] 客观分析大V情绪方向和强度，不预设其为反向指标
- [x] 区分「共识信号」和「极端情绪」，结合历史准确率判断参考价值
- [x] 不只总结「他说了什么」，还要解读「话外之音」

### 4. 历史镜鉴
- [x] 每次分析都要问：现在最像历史上的哪段行情？
- [x] 那次后来怎么走了？为什么这次可能不同？
- [x] 市场共识最容易在哪些地方出错？

### 5. 跨市场联动
- [x] 美股 → A股/港股的传导机制（ADR、北向资金、情绪）
- [x] 加密货币与纳斯达克的相关性变化
- [x] 美元/美债/VIX 对全球风险资产的压制

### 6. 资金面 ≠ 基本面
- [x] 短期波动更多由资金面（流动性、仓位、衍生品到期）驱动
- [x] 基本面的变化需要时间发酵，但市场的反应往往是瞬时的
- [x] 区分「短期交易逻辑」和「中期配置逻辑」
