# Memory 系统

此目录存储**用户个人数据**（会变化的、关于你是谁的信息），不存储分析方法论。

## 分工

| 存储位置 | 内容 | 变化频率 |
|----------|------|----------|
| `prompts/deep-research/` | **分析方法论**（怎么想）— 多空框架、掩护出货检测、反向阅读等 | 低（框架稳定） |
| `memory/` | **用户个人数据**（关于谁）— 持仓、观点、历史判断、偏好等 | 高（持续更新） |

## 当前记忆

（暂无用户个人数据。后续对话中当你表达新的仓位观点、市场判断、标的好恶时，AI 会自动写入。）

## 可能存储的内容类型

- **持仓与关注**：「我重仓了 MU，成本 $950」
- **市场世界观**：「我认为 AI 不是泡沫，但半导体估值透支了 2 年」
- **历史判断**：「6/7 我判断 SpaceX IPO 后会反弹——结果验证/打脸」
- **偏好设置**：「我更关注短期交易机会而非长期配置」
